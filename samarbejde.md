# Samarbejde på GitHub

Guide til hvordan man arbejder med filer online via github.

Information på GitHub er ordnet i repositories kaldet repo. Hvis man godt vil gøre et repository til sit eget kan bruge ```fork```. På den måde bliver der lavet en online kopi af til dig selv. Du kan nu ændre i filerne og det hele forbliver hos dig. 

Hvis det oprindelige repo bliver opdateres kommer der en besked og man kan vælge ```Sync fork``` så man kopiere det nye ind på ens eget.


Hvis man vil bidrage med noget skal man lave en ```Pull reguest``` så vil den som har hoved-repo kunne flette det ind.
